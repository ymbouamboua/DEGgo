# Generate DEGgo HTML/PDF Report

Creates a DEGgo analysis report from the results generated by
[`run_deggo()`](https://ymbouamboua.github.io/DEGgo/reference/run_deggo.md).

## Usage

``` r
generate_deggo_report(
  results,
  output_dir = results$output_dir,
  formats = "html",
  report_template = NULL
)
```

## Arguments

- results:

  DEGgo results object returned by
  [`run_deggo()`](https://ymbouamboua.github.io/DEGgo/reference/run_deggo.md).

- output_dir:

  Output directory.

- formats:

  Character vector. Supported values: `"html"` and `"pdf"`.

- report_template:

  Optional path to custom R Markdown template.

## Value

Invisibly returns generated report paths.
