# Run DEGgo pairwise differential expression analysis

Internal helper used by
[`run_deggo()`](https://ymbouamboua.github.io/DEGgo/reference/run_deggo.md)
to perform automated pairwise differential expression analyses using
DESeq2.

## Usage

``` r
.run_deggo_pairwise(
  counts,
  metadata,
  orgdb,
  dirs,
  ontology,
  padj_cutoff,
  logfc_cutoff,
  top_n_heatmap,
  top_n_labels,
  txtsize,
  pairwise_group_cols,
  pairwise_contrast_col,
  pairwise_contrasts,
  pairwise_mode,
  log
)
```

## Arguments

- counts:

  Integer count matrix.

- metadata:

  Sample metadata data frame.

- orgdb:

  Annotation database object.

- dirs:

  Output directory list generated by `.deggo_dirs()`.

- ontology:

  GO ontology ("BP", "MF", or "CC").

- padj_cutoff:

  Adjusted p-value cutoff.

- logfc_cutoff:

  Absolute log2 fold-change cutoff.

- top_n_heatmap:

  Number of top genes displayed in heatmaps.

- top_n_labels:

  Number of labeled genes in volcano plots.

- txtsize:

  Base text size used for plots.

- pairwise_group_cols:

  Metadata columns defining pairwise groups.

- pairwise_contrast_col:

  Column storing generated contrasts.

- pairwise_contrasts:

  Optional named list of pairwise contrasts.

- pairwise_mode:

  Pairwise comparison mode.

- log:

  DEGgo logging function.

## Value

A list containing pairwise differential expression results, significant
genes, plots, GO enrichment results and summary statistics.

## Details

The workflow includes:

- Pairwise contrast generation and testing.

- PCA analysis.

- DEG annotation.

- Volcano plot generation.

- Heatmap generation.

- GO enrichment analysis.

- Summary table generation.
