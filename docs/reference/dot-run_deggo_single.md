# Run DEGgo single-comparison analysis

Internal helper used by
[`run_deggo()`](https://ymbouamboua.github.io/DEGgo/reference/run_deggo.md)
to perform a standard differential expression analysis workflow for a
single contrast.

## Usage

``` r
.run_deggo_single(
  counts,
  metadata,
  method,
  design_formula,
  contrast,
  orgdb,
  dirs,
  ontology,
  padj_cutoff,
  logfc_cutoff,
  top_n_heatmap,
  top_n_labels,
  txtsize,
  log
)
```

## Arguments

- counts:

  Integer count matrix.

- metadata:

  Sample metadata data frame.

- method:

  Differential expression method.

- design_formula:

  Design formula passed to DESeq2, edgeR or limma.

- contrast:

  Contrast vector for differential expression analysis.

- orgdb:

  Annotation database object.

- dirs:

  Output directory list generated by `.deggo_dirs()`.

- ontology:

  GO ontology ("BP", "MF", or "CC").

- padj_cutoff:

  Adjusted p-value cutoff.

- logfc_cutoff:

  Absolute log2 fold-change cutoff.

- top_n_heatmap:

  Number of top genes displayed in heatmaps.

- top_n_labels:

  Number of labeled genes in volcano plots.

- txtsize:

  Base text size used for plots.

- log:

  DEGgo logging function.

## Value

A list containing differential expression results, significant genes,
plots, GO enrichment results and summary statistics.

## Details

The workflow includes:

- Differential expression analysis.

- Gene annotation.

- Significant DEG extraction.

- Volcano plot generation.

- PCA visualization (DESeq2 only).

- Heatmap generation.

- Gene Ontology enrichment analysis.

- Result export and summary generation.
